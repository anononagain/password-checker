# Password Strength Checker

A simple Python-based tool that checks whether your password is **Strong** or **Weak** using pattern-matching rules and regular expressions.  
Built as a beginner-friendly cybersecurity project. 

## Features

Checks for:
- Uppercase and lowercase letters  
- Numbers (0–9)  
- Special characters (@, #, $, %, etc.)  
- Minimum password length (8 characters)

Gives instant feedback — “Weak” or “Strong”  
Loops until you type `exit`  
Easy to run from terminal or VS Code  

***


Tech Used
- Python 3  
- `re` (Regular Expressions) module  
- Command-line interface
